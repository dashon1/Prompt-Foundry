
# PromptOS – ALL OF THE ABOVE
Includes:
- /bundle/json: prompt specs
- /schemas: TypeScript + Zod
- /edge/generate-prompt.ts: Supabase Edge starter
- /web: Minimal Next.js UI

## UI
cd web && npm install && npm run dev

## Supabase Edge
Create function `generate-prompt`, paste file, deploy.

## Extend
Add JSON to /bundle/json and wire inputs to forms.
